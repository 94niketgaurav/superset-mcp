Superset Chatbot

A smart SQL assistant for Apache Superset - Ask questions in natural language, get SQL queries

Installation

  1. Drag this button to your browser's bookmarks bar:
    SQL Chatbot
  2. Navigate to Superset in your browser (e.g., http://localhost:8088)
  3. Click the bookmarklet in your bookmarks bar to open the chatbot
  4. Start chatting! Ask questions like "Show me all users created last month"
Note
Make sure the chatbot server is running at http://localhost:5050 before using the bookmarklet.
Start it with: python chatbot/api_server.py

Custom Configuration

If your chatbot server runs on a different URL, update the configuration below:

javascript:(function(){var a='http://localhost:5050';if(window.SupersetChatbot){window.SupersetChatbot.open();return}var s=document.createElement('script');s.src=a+'/static/widget.js';s.onload=function(){if(window.SupersetChatbot){window.SupersetChatbot.init({apiUrl:a});setTimeout(function(){window.SupersetChatbot.open()},100)}};s.onerror=function(){alert('Failed to load Superset Chatbot. Make sure the chatbot server is running at '+a)};document.head.appendChild(s)})();

Features

  • Natural Language to SQL - Ask questions in plain English
  • Smart Table Detection - Automatically identifies relevant tables
  • Time Filter Support - Understands "last month", "this year", etc.
  • Schema Awareness - Works with your database schemas
  • Few-Shot Learning - Improves from your existing queries
  • Direct SQL Lab Integration - Opens queries directly in Superset
